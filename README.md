# Introduction 
Creditcoin Shared components are used by multiple projects in the Creditcoin ecosystem. 

# Getting Started
The Creditcoin Client NuGet packages are available on [NuGet.org](https://www.nuget.org/profiles/Creditcoin).

# Build and Test
```
dotnet restore
dotnet build
dotnet pack
```
